About the Facial Recognition System

Person identification and processing the information of the person by looking at the person's image is a tedious task if processed by a human. The earlier built systems for facial recognition are not robust and time-consuming. With computer vision algorithms growing rapidly in today's world, we plan to build a system that can perform facial recognition and process the information of the person using Neural Networks.

This application, when provided with the image of the audience, can recognize all the attendees in the image and store their information in the databases for later use. The models we are trying to build require very few samples as compared to earlier models, reducing training times and data requirements. In this project, we intend to create an AI Image recognition system to track the attendance of participants by analyzing facial features and marking their presence in the database.

Our system follows the process in the following steps:

  1. Image Acquisition
  2. Image Detection
  3. Attendance Registration
  4. Monitoring

We plan to build a neural network inspired by the latest Neural Network architectures available for computer vision tasks.
